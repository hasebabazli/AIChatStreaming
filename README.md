# AI Chat Token Streaming

Course: Advanced Web Development - Streaming

Student:  حسيب صلاح  ابازلي
## Project Description

This project demonstrates real-time AI chat token streaming using ASP.NET Core MVC and Server-Sent Events (SSE). Instead of displaying the response all at once, the server streams the response token-by-token in real time without reloading the page.

## Technologies Used

- ASP.NET Core MVC
- Razor Views
- JavaScript
- Server-Sent Events (SSE)
- Bootstrap

## Features

- Real-Time Token Streaming
- Adjustable Streaming Speed
- Stop Streaming
- Live Token Counter
- Connection Status
- Responsive User Interface

## Smart Streaming Idea

The user can control the streaming speed and stop the streaming process at any time.
